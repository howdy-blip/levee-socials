🐙 GitHub Sync

Store your dashboard data in a GitHub repo. Every change is automatically pushed — so whoever has this file (or a GitHub Pages URL) always loads the latest data.

Needs repo scope (or public_repo for public repos). Stored only in your local browser — never sent anywhere except GitHub.

💡 Tip — share a live URL: Put this HTML file in the same repo and enable GitHub Pages. Anyone who opens your Pages URL will always see the latest data automatically.
